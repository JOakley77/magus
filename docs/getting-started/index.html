Get Dependencies

The dependency list is small and kind of obvious. Bootstrap & jQuery is all you need to get started.

Bootstrap

Download latest source code

Get the original files for all CSS and JavaScript by downloading the latest version directly from GitHub.

Clone or fork via GitHub

Clone the entire project or fork your own version of Bootstrap to make it your own by visiting us on GitHub.

Install with Bower

Install and manage the original files for all CSS and JavaScript, along with a local copy of the docs, using Bower.

$ bower install bootstrap

What's included

Within the download you'll find the following directories and files, logically grouping common assets and providing both compiled and minified variations.

Once downloaded, unzip the compressed folder to see the structure of (the compiled) Plugin. You'll see something like this:

magus/
├── css/
│   ├── magus.css
│   ├── magus.min.css
├── js/
│   ├── magus.js
│   ├── magus.min.js

This is the most basic form of Magus: compiled files for quick drop-in usage in nearly any web project. We provide compiled CSS and JS (magus.*), as well as compiled and minified CSS and JS (magus.min.*).

Basic usage

In its simplest form, you can initialize Magus using the default options, or dive into a few examples we've started for you.

Copy and paste the HTML from below to get started with a bare bones Magus document.

<!DOCTYPE html>
<html>
    <head>
        <title>Magus Example</title>
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <!-- Dependencies -->
        <link href="css/bootstrap.min.css" rel="stylesheet" media="screen">
    </head>
    <body>
        <h1>Hello, world!</h1>

        <!-- JavaScript plugins (requires jQuery) -->
        <script src="http://code.jquery.com/jquery.js"></script>
        
        <!-- Include all compiled plugins (below), or include individual files as needed -->
        <script src="js/bootstrap.min.js"></script>
        
        <!-- Include the Magus Plugin -->
        <script src="js/magus.min.js"></script>
    </body>
</html>

Browser compatibility

Bootstrap is built to work best in the latest desktop and mobile browsers. Magus should match functionality at the same level as Bootstrap.

Supported browsers

Specifically, we support the latest versions of the following:

  • Chrome (Mac, Windows, iOS, and Android)
  • Safari (Mac and iOS only, as Windows has more or less been discontinued)
  • Firefox (Mac, Windows)
  • Internet Explorer
  • Opera (Mac, Windows)

For more information on browser compatibility you should view the Bootstrap Documentation.
